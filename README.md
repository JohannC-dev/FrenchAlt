## Development

### Getting Started

**System Requirements**

* [Node.js][nodejs] v12
* [MongoDB database][mongolink]

---

**Clone and Install Dependencies**

```console
> git clone https://github.com/JohannC-dev/FrenchAlt.git
> cd FrenchAlt
> npm install
```

---

**Launch Bot**

```console
> node app.js
```
